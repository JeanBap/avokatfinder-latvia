What is Kadastrs?

The Latvian State Cadastre — the official register of land parcels and buildings maintained by the State Land Service, providing the technical description of each property.

Key Facts — Kadastrs
  • Maintained by Valsts zemes dienests (State Land Service)
  • Every property has a unique kadastra apzīmējums (cadastral code)
  • Cadastral value used to calculate property tax
  • Must be consistent with Zemesgrāmata ownership entry
  • Public portal: kadastrs.lv

The Latvian kadastrs (cadastre) is maintained by the Valsts zemes dienests (State Land Service). It records the physical and technical characteristics of all land parcels and buildings in Latvia: area, boundaries, construction details, cadastral value, and use category.

Every property in Latvia has a unique cadastral designation (kadastra apzīmējums), a code that links the physical cadastral record to the legal ownership record in the Zemesgrāmata. The Zemesgrāmata entry references the cadastral designation, so both registers must be consistent.

The cadastral value (kadastrālā vērtība) is the State Land Service's official assessment of a property's value. It is used for calculating property tax (nekustamā īpašuma nodoklis) and for determining the minimum stamp duty threshold on property transfers.

Buyers should verify that the cadastral record matches what the seller represents about the property — particularly the registered area, the number and type of buildings, and the land use category. Discrepancies can indicate illegal construction, encroachments, or planning restrictions.

Cadastral information is publicly searchable via the State Land Service portal (kadastrs.lv). A full cadastral extract (kadastra izziņa) is available for a small fee and shows all technical details including the cadastral map.

Common Mistake: Relying solely on the seller's description of property size. Always verify the registered cadastral area, which is the legally binding measurement. A property advertised as 150m² may have only 120m² in the cadastre.
Expert Tip: Check the land use category in the kadastrs before buying. Residential, agricultural, forest, and commercial categories have different planning permissions and tax rates. Converting land use category requires a local government decision and can be lengthy.
Related terms: ZEMESGRĀMATA PIRKUMA LĪGUMS NEKUSTAMĀ ĪPAŠUMA NODOKLIS

Frequently Asked Questions

What is the cadastral value used for?

Property tax (nekustamā īpašuma nodoklis) is calculated as a percentage of the cadastral value. It is also used to set the minimum transaction value for stamp duty purposes.

Can the cadastral value be challenged?

Yes. If you believe the cadastral value is incorrectly assessed, you can apply to the State Land Service for a review. An independent appraisal can support the challenge.

Do I need to update the cadastre after renovation?

Yes. If you make structural changes to a building — extensions, conversions, demolitions — you must update the cadastral record to reflect the new state. Failure to do so can complicate future sales.
